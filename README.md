## 🔧 3 分钟极速复刻，立即上线

真正的手把手、零门槛部署方案。

1. `git clone` 或直接下载本仓库源码至本地目录。
2. 打开 [微信官方开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
3. 选择“导入项目”，选中 `SBTI-MiniProgram` 文件夹。
4. 在 `project.config.json` 替换为你自己的 AppID。
5. **在代码中全局搜索“广告位”，一键填入你的流量主 ID**。
6. 点击上传，提交微信官方审核。恭喜你，你已成为这款日活神器的新操盘手！
